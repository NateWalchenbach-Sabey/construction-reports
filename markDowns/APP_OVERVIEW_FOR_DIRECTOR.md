# Sabey Construction - Project Management & Reporting System

## Executive Summary

The **Sabey Construction** application is a comprehensive digital platform designed to streamline project management, weekly reporting, and construction oversight across all Sabey projects. This system replaces manual processes with an intuitive, web-based solution that provides real-time visibility into project status, budgets, timelines, and workforce activity.

---

## What This System Does

The application serves as your **central command center** for construction project management, allowing you to:

- **Track all projects** across multiple regions in one unified dashboard
- **Monitor budgets and costs** with real-time variance tracking
- **Review weekly reports** from superintendents and project managers
- **Analyze workforce data** and subcontractor activity
- **Generate executive summaries** and printable reports
- **Get instant answers** using the built-in AI assistant (Sabey AI)

---

## Key Features

### 1. **Project Dashboard**

**What it shows:**
- All active projects organized by region (Ashburn, Seattle, Austin, Columbia, Quincy, Manhattan, Umatilla, Butte, and Non-SDC)
- Project codes, names, and current status
- Budget information (Original Budget vs. EAC - Estimated At Completion)
- Budget variance indicators (green for on-track, red for over-budget)
- Percentage complete for each project
- Latest report dates and workforce counts

**Why it matters:**
At a glance, you can see which projects are on track, which need attention, and how resources are being allocated across all regions.

---

### 2. **Project Details & Timeline**

**What it shows:**
- Complete project information including start dates, scheduled completion, and tenant details
- Visual timeline showing project duration and how it compares to other projects
- Budget vs. EAC charts showing financial performance over time
- Workforce trends showing trade worker counts week-over-week
- All weekly reports for the project in chronological order

**Why it matters:**
Deep dive into any project to understand its full history, current status, and trajectory. The timeline view helps identify scheduling conflicts and resource allocation patterns.

---

### 3. **Weekly Report Management**

**Creating Reports:**
- Superintendents and PMs can create new weekly reports directly in the system
- Reports capture:
  - Work performed during the week
  - Safety information and incidents
  - Subcontractor activity by trade (Electrical, Plumbing, HVAC, etc.)
  - Number of trade workers on site
  - Project staff assignments
  - Architect information

**Viewing Reports:**
- Once submitted, reports become read-only for historical accuracy
- Easy navigation to view past reports
- Reports are linked to specific projects and dates

**Why it matters:**
Standardized reporting ensures consistent data collection across all projects. Historical reports provide an audit trail and help identify trends.

---

### 4. **Weekly Report Status**

**What it shows:**
- Which projects have reports submitted for the current week
- Which projects are missing reports
- Visual indicators (green checkmarks for complete, red alerts for missing)
- Ability to filter by region or date range

**Why it matters:**
Ensures accountability and helps identify which project teams need reminders to submit their weekly updates.

---

### 5. **Weekly Summary Reports**

**What it provides:**
- Comprehensive weekly summaries across all projects
- Executive-level overview of activities, budgets, and workforce
- Printable format for meetings and distribution
- PDF export capability for archiving and sharing

**Why it matters:**
Provides a consolidated view of the entire operation for executive review and stakeholder communication.

---

### 6. **Sabey AI Assistant**

**What it does:**
- Answers questions about projects, budgets, reports, and data
- Provides instant insights without needing to navigate multiple screens
- Understands construction terminology and project context
- Can analyze trends, compare projects, and summarize information

**Example questions you can ask:**
- "Which projects are over budget?"
- "Show me all reports from last week for Seattle projects"
- "What's the total workforce across all active projects?"
- "Compare budget variance between ASH-DC-12 and SDC-SEA-53"

**Why it matters:**
Saves time by providing instant answers instead of manually searching through data. Acts as a knowledgeable assistant that understands your construction operations.

The system supports different user roles with appropriate access levels:

- **Superintendents:** Can create and manage reports for assigned projects
- **Project Managers (PMs):** Full project oversight and reporting capabilities
- **Executives:** Read-only access to all projects with executive summary views
- **Administrators:** Full system access including user management

## How It Works (Simple Workflow)

### For Superintendents/PMs:

1. **Log in** to the system
2. **Select a project** from the dashboard
3. **Click "New Report"** to create a weekly report
4. **Fill in the details:**
   - Work performed
   - Safety information
   - Subcontractor activities
   - Trade worker counts
5. **Click "Upload Report"** to save
6. Reports are automatically linked to the project and become part of the historical record

### For Executives/Directors:

1. **Log in** to the system
2. **View the dashboard** to see all projects at a glance
3. **Click on any project** to see detailed information
4. **Review weekly summaries** for consolidated reports
5. **Use Sabey AI** to ask questions and get instant insights
6. **Export reports** as PDFs for meetings and documentation
